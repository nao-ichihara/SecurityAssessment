# Security Assessment
- Security Assessment は NIST SP 800-53, CIS Controls, ISO 20071, NIST CSF, PCI-DSS などのセキュリティ成熟度評価のためのフレームワークに準じたアセスメントを行うアプリケーションである
- /docs フォルダに、要件定義書、設計書、テスト仕様書、新たにフレームワーク追加のためのプロンプトが置いてある
- 主な機能
  - フレームワークの選択
  - 要求事項に対応してカテゴリ別の質問項目の表示、回答選択、メモ記入欄
  - 評価結果に基づく総合スコア、成熟度レベル、回答率、レーダーチャート、セクション別スコアなどのダッシュボード表示
  - ダッシュボード上の要対応事項、改善推奨事項の自動表示と、総合評価の自由記入欄
  - 評価結果のレポート出力
  - 回答内容のファイル保存と読み込み(.json形式)
  - 解説書の表示
